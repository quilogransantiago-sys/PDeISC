🔄 forEach() – Recorriendo arrays

Ejecuta una función por cada elemento del array.

1️⃣ Mostrar saludo para cada nombre

Nombres: ["Ana","Carlos","Lucía","Miguel"]

2️⃣ Imprimir el doble de cada número

Números: [3, 7, 12, 5]

3️⃣ Mostrar nombre y edad de cada persona

Personas: [{"nombre":"Ana","edad":25},{"nombre":"Luis","edad":30},{"nombre":"Eva","edad":22}]

💡 forEach() itera sobre cada elemento. Útil para mostrar datos o hacer cálculos.
