splice() paso a paso

Modifica el array directamente: elimina, inserta o reemplaza.

1️ Eliminar dos elementos desde la posición 1

letras.splice(1, 2) → Empieza en índice 1 y borra 2 elementos.

Array actual: ['A','B','C','D','E']

2️ Insertar un nombre en segunda posición

nombres.splice(1, 0, "Nuevo") → En índice 1, elimina 0 e inserta.

Array actual: ['Ana','Carlos','Luisa']

3️ Reemplazar dos elementos desde la posición 2

colores.splice(2, 2, "X", "Y") → En índice 2, borra 2 y mete dos nuevos.

Array actual: ['Rojo','Verde','Azul','Amarillo','Naranja']
