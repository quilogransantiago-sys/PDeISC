slice() – Copiar sin modificar

slice() crea un nuevo array con una parte del original. El original no se toca.

1️ Copiar los primeros 3 elementos

numeros.slice(0, 3) → Desde índice 0 hasta el 3 (no incluye índice 3).

Array original: [10, 20, 30, 40, 50, 60]
Copia (primeros 3): —

2️ Copiar desde posición 2 hasta la 4

peliculas.slice(2, 5) → Desde índice 2 hasta el 5 (exclusivo). Así obtenemos las posiciones 2,3,4.

Array original: ["El Padrino","Titanic","Inception","Gladiador","Matrix"]
Copia (posiciones 2 a 4): —

3️ Últimos 3 elementos (del mismo array de películas)

peliculas.slice(-3) → Con números negativos empieza desde el final.

El mismo array original: ["El Padrino","Titanic","Inception","Gladiador","Matrix"]
Copia (últimos 3): —
