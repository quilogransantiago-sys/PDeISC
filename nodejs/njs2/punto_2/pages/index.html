🌐 Servidor con módulos HTTP y File System

Bienvenido

Este sitio es servido por un servidor Node.js que utiliza los módulos nativos http y fs.

El archivo index.html se lee del sistema de archivos y se envía al navegador.

Los estilos CSS y el script JavaScript también son leídos mediante fs.readFile y enviados con el tipo MIME correcto.

Características técnicas

  • Servidor HTTP puro (sin Express).
  • Manejo de rutas y tipos MIME.
  • Modo claro/oscuro persistente con localStorage.
  • Diseño responsivo (sin anchos fijos, adaptado a móvil).
  • Código 100% en español (comentarios, variables, textos).

Desarrollado con Node.js · Módulos http y fs
