Acerca del proyecto

Trabajo práctico para la materia Programación de Interfaces de Sistemas Computacionales.

Tecnologías utilizadas:

  • Node.js con módulos ES6
  • Módulos nativos: http, fs, url
  • Paquete NPM: upper-case
  • API de clima: Open-Meteo

Objetivo: Demostrar el uso de módulos propios, servidor HTTP, sistema de archivos, análisis de URL y paquetes externos.
